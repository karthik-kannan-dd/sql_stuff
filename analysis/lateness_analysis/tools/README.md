# Dasher Lateness Lookup Tool

A Streamlit app to look up lateness patterns for individual dashers.

![Screenshot](screenshot.png)

## Setup

```bash
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

## Usage

```bash
streamlit run app.py
```

Enter a Dasher ID to see their delivery lateness history, summary stats, and charts.
